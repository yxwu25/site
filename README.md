## How to

To run app locally you need to clone repo and execute several cmds in the shell:

```bash
npm install
```

and also:

```bash
bower install
```

Don't forget to start your local mongo before starting app.

### Requirements

- OS X, Linux, Windows;
- It's tested to run with node v0.8 and v0.10;
